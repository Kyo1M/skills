# プラットフォーム別の体裁

note と Zenn で記事の体裁・frontmatter・画像の方針を切り替える。記事 frontmatter の `platform` 値を見て判断する。

---

## note

ビジネス・読み物・考え方の言語化向け。

### 文章

- 分量は字数で縛らない。**内容が過不足なく伝わる長さ**を優先し、薄い水増しも詰め込みすぎも避ける
- 見出しは **H2（##）と H3（###）のみ**
- 太字は重要なメッセージに限定（1 段落に 1 箇所以下）
- 外部リンクは必要最小限

### セクションごとの分量バランス

字数ではなく、セクションの役割に対する相対的な厚みで考える。

| セクション | 分量 | ポイント |
|-----------|------|---------|
| 導入（リード文） | 軽 | 読者の課題に共感し、読むメリットを示す。冒頭 3 行で掴む |
| 問題提起・背景 | 中 | 「なぜ重要か」を丁寧に説明。読者が体感済みなら短く |
| 本論（各 H2） | 厚 | 主張→理由→具体例で厚みを持たせる。記事の主役 |
| 具体例・事例 | 中 | 状況を描写し読者がイメージできるように |
| まとめ・CTA | 軽 | 要点を再確認し次のアクションを促す |

### note プラットフォームの考慮事項

- **冒頭 3 行が勝負**: スマホでは最初の 3 行しか見えない。ここで読者を掴む
- **目次**: H2 見出しが自動で目次になる。見出しは端的に
- **スキマ読み対策**: 見出しと太字だけで要点が伝わるように
- OGP 画像（サムネイル）は別途考慮（本スキルでは扱わない）

### 画像

コンサルスライド風（白背景、青とグレー基調、日本語テキスト入り、16:9）。`image-prompts.md` の図解パターンを使う。

### frontmatter

管理用 frontmatter（`templates/article.md`）をそのまま使う。note には機械可読な frontmatter 規格はないため、公開時は本文のみをコピーする。

---

## Zenn

技術記事・実装知見・コード中心向け。

### 文章

- 文字数の縛りは緩め。内容の正確さと再現性を優先する
- **コードブロックは言語指定必須**（` ```ts ` など）
- 手順は具体的に。コマンド・バージョン・前提環境を明記する
- 見出しは H2 / H3 中心。Zenn は H1 を本文に使わない（タイトルは frontmatter）
- 専門用語は初出時に簡潔な説明か正式名称を添える

### 画像

図解またはスクリーンショット。コンサルスライド風にこだわらず、技術的に正確で見やすいものを優先する。アーキテクチャ図・フロー図は `image-prompts.md` のパターンを流用可。

### frontmatter（公開準備時に生成）

Zenn は記事ファイル先頭に専用 frontmatter を要求する。公開準備フェーズで管理用 frontmatter から次を生成する。

```yaml
---
title: ""           # 記事タイトル
emoji: "📝"          # アイキャッチの絵文字 1 文字
type: "tech"        # tech（技術記事） | idea（アイデア記事）
topics: []           # トピック（タグ）。例: ["typescript", "nextjs"]
published: false     # true で公開
---
```

- `type` はコード・実装中心なら `tech`、考察・ポエム寄りなら `idea`
- `topics` は管理用 frontmatter の `tags` から変換する
- `published` は公開タイミングで `true` にする
- 管理用 frontmatter（`slug` / `status` など）は Zenn には載せない。`published/` 配下の記事フォルダでは管理用と Zenn 用のどちらを正とするかを決めて運用する（推奨: Zenn 形式を本文先頭に、管理メタは別途台帳かコミットメッセージで残す）

---

## 共通

- slug は kebab-case の英小文字
- 画像は記事フォルダ内の `assets/` に置き、本文から `![alt](assets/NN-desc.png)` で参照する
- トーン（敬体/常体）は記事内で統一する
- 字数は手段であって目的ではない。目標字数に届かせるための水増しも、収めるための無理な圧縮もしない
